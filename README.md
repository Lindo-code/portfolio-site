## 🖼️ Preview

A premium, modern **developer portfolio** built to showcase real-world work, open-source activity, and technical capabilities with performance and user experience at the core.

<br>

## ✨ Overview

This portfolio is designed as a **high-performance, visually refined platform** for presenting:

- Real projects (not just concepts)
- Open-source contributions (live GitHub stats)
- Technical stack and capabilities
- Client testimonials
- Contact and engagement pathways

It emphasizes **clarity, motion, and usability** without compromising performance — especially on low-end devices.

<br>

## 🚀 Features

### 🎯 Core

- Dynamic project showcase with horizontal scrolling
- Open-source activity dashboard (GitHub API + caching)
- Interactive testimonials ticker
- Video-based hero section
- Fully responsive design

### ⚡ Performance Focus

- SSR-first architecture (Next.js)
- Lazy-loaded sections (code splitting)
- Local caching for API data
- Optimized animations (Framer Motion)

### 🎨 UI/UX

- Premium dark aesthetic
- Glass + subtle gradients
- Smooth motion transitions
- Half-pill hero image reveal
- Clean typography and spacing

<br>

## 🛠️ Tech Stack

### Frontend

- Next.js (App Router)
- React
- Framer Motion
- CSS (custom, no UI frameworks)

### Backend / Data

- Cloudflare Workers (API proxy + validation)
- GitHub API
- LocalStorage caching strategy

### Tooling

- Vercel (deployment)
- Cloudflare (CDN / routing)
- Microlink (link previews)

<br>

## 🧠 Architecture Highlights

- **Component-based modular structure**
- **Lazy-loaded sections** for performance
- **Animation system using Framer Motion variants**
- **Client-side caching strategy** to reduce API calls
- **Separation of UI and logic**

<br>

## 🌍 Deployment

- Frontend: Vercel
- Domain/CDN: Route53
- Backend APIs: Cloudflare Workers or custom VM
